Football Spending & Performance Analytics (MUST DOWNLOAD FILE TO VIEW)

Overview
This project analyzes the relationship between football club spending and on-field performance.

Tools Used
- R
- R Shiny
- Tableau

Features
- Interactive dashboards
- Animated visualizations
- Statistical analysis
- Data storytelling

Key Findings
- Spending contributes to performance but is not the sole determinant of success.
- Organizational stability and efficiency also play significant roles.
